Layout concepts

Four ways to make the dashboard feel like an AI product, not a CRM.

Each concept keeps the current colours and wordmark but replaces the nine equal boxes below Leads & Signals with a layout that leads with insight and makes the tools feel like things you ask an assistant to do. Open any one full-screen.

01Conversational

Agentic Command Console

A single "ask anything" command bar is the hero; the six tools become AI plays you invoke, not forms. Context collapses to a slim rail. Feels like a copilot at Sara's desk.

02Product app

App Workspace

Tools and context move into a persistent left navigation — no grid at all. The canvas stays focused on the feed; a play opens as a focused work surface. Reads like software you log into.

Open full mockup →
03Editorial

Editorial Daily Briefing

A calm single-column note written by an AI analyst. The six tools shrink to one slim play toolbar; context becomes a tabbed strip. Premium, low-clutter, something you read with coffee.

Open full mockup →
04Bento

Bento Intelligence

Keeps cards but breaks the uniform grid into an asymmetric bento with real hierarchy — a hero "Top moves" tile, big-number stats, and tools sized by importance. Modern, never a contact list.
