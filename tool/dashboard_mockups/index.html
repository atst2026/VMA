Dashboard layout mockups

Five different ways to arrange the same panels. Pick one, or take pieces of several. None are wired up — these are visualisation only.
1

Sticky Sidebar Workspace

Permanent left rail (260px) with section nav, mini Top-3 always visible, Daily Refresh pinned. Main column scrolls. Linear / Notion feel.
→
2

Tabbed Sections

Five top tabs: Today / Pipeline / Candidates / Tools / Archive. Each shows only its 3–4 relevant panels. Big Top-3 pinned above every tab. Dramatically less on screen at once.
→
3

Accordion Hierarchy

Minimalist. Huge Top-3 hero at the top, everything else collapsed by default. Sara expands what she wants. Maximum focus, zero clutter. Apple Reminders / Things 3 feel.
→
4

Bento Grid

12-column asymmetric grid. Top-3 takes a wide hero block + two coloured stat tiles. Signal panels in equal-ish squares below. Action tools in a 4-card band at the bottom. Apple product-page energy.
→
5

Action Rail + Scroll Feed

Right-side sticky rail (320px) with Top-3 + one-tap quick tools always pinned. Main column is a vertical scroll-feed of signal panels. Works exactly like a news feed.
→
