Dashboard layout mockups · AI-product directions

Five layouts modelled on real AI products — each is a self-contained HTML file showing how the VMA dashboard would feel in that aesthetic. None are wired up; these are visual exploration so you can pick a direction.
1

Gemini 3.5 The current direction, taken further

Pale-blue radial halo hero with the day's brief written as a single sentence ("you have three calls worth making before lunch"). Top-3 cards on the halo, clean white panels below. Pulse-green live dot, JetBrains Mono labels, generous whitespace.

2

Claude / Anthropic Reading-led, narrative

Warm cream background, serif headings, document-like single-column reading layout. Each lead reads as a paragraph the AI has written for you, not a row in a table. Inline action buttons embedded in the prose.

3

ChatGPT / OpenAI Conversation thread

Left sidebar of past briefs (chat-history style). Main column is a back-and-forth: Sara asks "what's worth my time today?" and the AI replies with numbered intel + action chips. Composer at the bottom for ad-hoc queries.

4

Perplexity Dark search + cited answer

Dark navy background, cyan accents. Today's brief framed as a question ("what three calls before lunch?") with an answer card carrying numbered citations and a grid of source pills. Follow-up suggestion chips below; sticky search bar at the bottom.

5

Linear / Raycast Dim, glassmorphic, keyboard-first

Near-black background with soft indigo/pink/cyan gradient glows. Glassmorphic frosted panels. Left sidebar with keyboard-shortcut chips (G T / G L / G P). ⌘K command bar across the top. Top-3 in a glowing indigo-tinted hero card.
